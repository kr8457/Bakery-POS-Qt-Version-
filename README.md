# Bakery POS (Qt Version)

A modern and lightweight **Bakery Point of Sale (POS)** desktop application built using **Qt (C++ & Qt Widgets)**.  
The system provides product management, categories, billing, and cart functionality suitable for bakery shops and small retail stores.
